Class Counter

Provides thread-safe operations around a signed long integer.

Remarks

While Counter instances can be converted to/read as a long integer value, correct usage for multi-threading requires using the return value of the instance methods to obtain the accurate counter value. Treating the counter as a long integer value is only useful where an inaccurate, at a moment in time value, is ok such us updating a display with the 'current' value.
