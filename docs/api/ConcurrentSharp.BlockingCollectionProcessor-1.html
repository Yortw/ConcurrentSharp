Remarks

This class also exposes a static method called ProcessItems(IEnumerable<T>, Int32, Action<T>, ExpectedThreadLifetime) which can be used to process all the items in any , encapsulating creating and managing the blocking collection as well as processing the items.

To use this class, create or obtain a reference to the blocking collection you want to process. Create a new instance of BlockingCollectionProcessor<T> passing it the collection and other configuration information. Then add items to the blocking collection as required, they will automatically be processed. When you've finished and will no longer be adding any more items to the collection call WaitForCompletion() and then Dispose() on the processor. Any items already in the blocking collection when the processor is created will also be processed.

Examples
//Create a collection to contain items that need processing
var myCollection = new BlockingCollection<Customer>();

// Create a processor that only processes up to 4 items at a time, is short lived and uses a method called
// 'DoWork' to process items.
var myProcessor = new BlockingCollectionProcessor<Customer>(collection, 4, DoWork, ExpectedthreadLifetime.Short);

foreach (var customer in customerRepository.GetAll())
{
    myProcessor.Add(customer);
}

myProcessor.WaitForCompletion();
myProcessor.Dispose();
