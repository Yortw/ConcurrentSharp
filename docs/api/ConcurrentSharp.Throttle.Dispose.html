Method Dispose

Dispose()

Disposes this throttle and all internal resources.

Declaration
public void Dispose()
Implements
System.IDisposable.Dispose()
Remarks

Will cause any currently pending or future calls to Enter() to throw a System.ObjectDisposedException.

Does not stop any pending operations.
