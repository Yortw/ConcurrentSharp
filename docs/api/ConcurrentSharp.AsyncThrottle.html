Class AsyncThrottle

Instances of the AsyncThrottle class can be used to restrict the number of simultaneous operations taking place using the same throttle.

Inheritance
System.Object
AsyncThrottle
Namespace: ConcurrentSharp
Assembly: cs.temp.dll.dll
Syntax
public sealed class AsyncThrottle : IDisposable
Remarks

To use a AsyncThrottle create an instance specifying the maximum number of concurrent operations you want to occur. Then have each operation call Enter() when it begins and capture the returned token. When the operation completes, call ConcurrentSharp.ThrottleToken.Dispose(). The Throttle will ensure no more than the specified number of operations run at the same time.

Examples
var throttle = new AsyncThrottle(4); //No more than 4 simultaneous operations.

// Start 100 calls to a 'DoWork' function.
for (int cnt = 0; cnt < 100; cnt++)
{
    System.Threading.ThreadPool.QueueUserWorkItem
    (
        (reserved) => 
        {
            using (var token = await throttle.EnterAsync()) //Obtain a token from the throttle instance to restrict the concurrent jobs
            {
                DoWork(throttle);
            } // Token will be disposed here, allowing another job to run if one is waiting.
        }
    );
}
