Provides synchronisation between multiple threads, restricting concurrent operations to a maximum limit.

Instances of BlockingCollectionProcessor<T> are used to manage threads for processing instances.

CancellableTaskSource<T>

Combines a and a CancellationToken so the task can be cancelled via the token, while ensuring all resources are correctly cleaned up regardless of how the task completes.

Counter

Provides thread-safe operations around a signed long integer.

ReleaseToken

A token that when disposed releases a lock or other associated synchronisation primitive.

TaskExtensions

Extensions for System.Threading.Tasks.Task instances.

TaskFrom

Throttle

Instances of the Throttle class can be used to restrict the number of simultaneous operations taking place using the same throttle.

ThrottleToken

An object returned by Enter() that is used to notify the parent Throttle instance the job owning this token has completed.
