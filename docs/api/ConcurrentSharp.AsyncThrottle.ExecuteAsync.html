Method ExecuteAsync

ExecuteAsync(Action)

A convenience method that executes the action operation using the throttle, eliminating manual management of the throttle token.

Declaration
public Task ExecuteAsync(Action action)
Parameters
Type Name Description
Action action

The job to execute via the throttle.
Returns
Type Description
Task
Examples
var throttle = new Throttle(4); //No more than 4 simultaneous operations.

// Start 100 calls to a 'DoWork' function.
for (int cnt = 0; cnt < 100; cnt++)
{
    System.Threading.ThreadPool.QueueUserWorkItem
    (
        async (reserved) =>
        {
            await throttle.ExecuteAsync(() => DoWork()); // Execute the action via the throttle's Execute method.
        }
}

ExecuteAsync<TResult>(Func<TResult>)

A convenience method that executes the func operation using the throttle and returning it's result while eliminating manual management of the throttle token.

Declaration
public Task<TResult> ExecuteAsync<TResult>(Func<TResult> func)
Parameters
Type Name Description
Func<TResult> func

The function to execute via the throttle.
Returns
Type Description
Task<TResult>

The value returned by the func parameter.
Type Parameters
Name Description
TResult

The type of value returned by func.
Examples
var throttle = new Throttle(4); //No more than 4 simultaneous operations.

var results = new List*lt;int>();
// Start 100 calls to a 'DoWork' function.
for (int cnt = 0; cnt < 100; cnt++)
{
    System.Threading.ThreadPool.QueueUserWorkItem
    (
        (reserved) =>
        {
            var result = await throttle.ExecuteAsync(DoWorkAsync()); 
            lock (results)
            {
                results.Add(result);
            }
        }
}
