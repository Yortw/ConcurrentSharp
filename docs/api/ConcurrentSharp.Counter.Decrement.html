Method Decrement

Decrement()

Decrements the counter by one and returns the value of the counter immediately after the increment, as an atomic, thread-safe operation.

Declaration
public long Decrement()
Returns
Type Description
System.Int64

The value of the counter immediately after the decrement occurred.
Remarks

If the value of the counter underflows the System.Int64.MinValue then the counter value will wrap around to System.Int64.MaxValue.

Examples
var newValue = counter.Decrement();

Decrement(Int64)

Decrements the counter by delta and returns the value of the counter immediately after the decrement, as an atomic, thread-safe operation.

Declaration
public long Decrement(long delta)
Parameters
Type Name Description
System.Int64 delta

The amount to decrement the counter by.
Returns
Type Description
System.Int64
Remarks

If delta is zero, no change occurs. If delta is negative, this is equivalent to calling Increment(Int64).

If the value of the counter underflows the System.Int64.MinValue then the counter value will wrap around to System.Int64.MaxValue.

Examples
var newValue = counter.Decrement(5);
