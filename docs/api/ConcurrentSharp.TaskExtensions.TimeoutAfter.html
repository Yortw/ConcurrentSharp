Method TimeoutAfter

TimeoutAfter<T>(Task<T>, Int32)

Waits for the specified task to complete, or a timeout to occur, whichever happens first.

Declaration
public static Task<T> TimeoutAfter<T>(this Task<T> task, int timeoutMilliseconds)
Parameters
Type Name Description
Task<T> task

The task to wait for.
System.Int32 timeoutMilliseconds

The maximum time in milliseconds to wait. A value of zero or less than -1 will cause an immediate timeout. Specify System.Threading.Timeout.Infinite or -1 to indicate no timeout.
Returns
Type Description
Task<T>

A value of type {T} returned by task.
Type Parameters
Name Description
T

The type of value returned by the task.
Remarks

If a timeout occurs task will not be cancelled and will still run to completion.

Exceptions
Type Condition
System.Threading.Tasks.TaskCanceledException

Thrown if task is cancelled.
System.TimeoutException

Thrown if a timeout occurs before task completes.
